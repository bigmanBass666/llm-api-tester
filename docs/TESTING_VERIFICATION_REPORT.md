## ⚠️ 已知局限

1. **并发测试未实现**: `concurrency` 参数在 `test_all` 中标记为"暂未实现"
   - 影响: 无法测试高并发场景下的性能
   - 建议: 后续使用 `asyncio.Semaphore` 实现

2. **配置文件依赖**: 元数据从 `platforms.yaml` 加载
   - 如果文件缺失，并发数等信息为空
   - 已有警告提示，不影响基本功能

3. **单平台测试**: 目前仅智谱AI有完整实现
   - 框架设计可扩展，但需子类实现
